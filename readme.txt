== Description ==

<strong>افزونه باشگاه مشتریان وردپرس</strong> به شما اجازه می دهد که شماره موبایل مشتریان و کاربران سایتتان را در دفترچه تلفن <a href="http://farazsms.com/">سامانه پیامک فراز اس ام اس</a> ذخیره کرده و هر زمان که خواستید برای مشتریان سایت و کاربران سایت اس ام اس مارکتینگ هوشمندانه انجام دهید.

افزونه باشگاه مشتریان وردپرس به دفتر تلفن سامانه پیامکی فراز اس ام اس متصل شده و زمانی که شخصی شماره مبایل خود را جهت عضویت در <a href="https://farazsms.com/digits-wordpress-mobile-number-signup-and-login/">افزونه دیجیتز</a> یا خرید در <a href="https://fa.wordpress.org/plugins/woocommerce/">افزونه فروشگاه ساز ووکامرس</a> وارد می کند شماره موبایل به صورت اتوماتیک در دفترچه تلفن ذخیره می شود.

توجه: برای استفاده از افزونه باشگاه مشتریان وردپرس حتما می بایست از <a href="https://farazsms.com/ippanel/">ippanel</a> سامانه پیامکی داشته باشید.
